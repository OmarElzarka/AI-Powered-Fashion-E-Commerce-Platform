# AI-Powered Fashion E-Commerce Platform

A full-stack multi-vendor fashion marketplace built with **ASP.NET Core** and **Angular**. The platform enables sellers to manage products and customers to browse, purchase, and track orders.

## ✨ Features

- Multi-vendor marketplace
- Customer & Seller authentication
- Product, Category, and Inventory Management
- Shopping Cart & Order Management
- AI Stylist for personalized outfit recommendations
- AI Shopping Agent that selects matching products, adds them to the shopping cart, and explains its decisions
- Responsive and modern user interface

## 🛠️ Tech Stack

- ASP.NET Core
- Angular
- SQL Server
- Entity Framework Core
- JWT Authentication
- RESTful APIs

## 🚀 AI Features

### AI Stylist
Provides personalized outfit recommendations based on:
- User preferences
- Personality
- Occasion
- Season

### AI Shopping Agent
Users can describe what they want, for example:

> "Choose a casual summer outfit under $150."

The AI agent:
1. Selects matching products from the marketplace.
2. Creates a complete outfit.
3. Adds the selected items to the shopping cart.
4. Explains why each item was chosen.
